# 🔥 Smart Firefighting System Using Robot

## 📌 Overview
An intelligent firefighting robot designed for early fire detection and emergency response using AI, IoT, and embedded systems.

---

## 🚀 Features
- Fire detection
- Smoke detection
- Autonomous robot movement
- Alarm activation
- Obstacle avoidance
- Real-time monitoring
